About This Project

MP BTech Lateral Entry College Predictor is my major project designed to help diploma students estimate their admission chances into second-year B.Tech programs across Madhya Pradesh.

Project Overview

This platform analyzes historical counselling cutoff data and provides tools such as College Predictor, Rank Predictor, and College Search.

The goal is to simplify the counselling process by giving students quick, data-driven insights based on previous years' admission trends.

Technology Stack

Backend

Python, Flask, SQLAlchemy

Frontend

HTML, CSS, Jinja2 Templates

Database

SQLite

Data Processing

Pandas and rule-based matching logic

Data Source

The prediction system is built using publicly available counselling data released by the Directorate of Technical Education (DTE), Madhya Pradesh.
